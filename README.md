# Climate Data Portal
The Climate Data Portal is a web-based application designed to streamline the process of collecting, processing, and visualizing climate data from various sources.
## Features
* Automated data ingestion from CSV, JSON, and NetCDF files
* Data visualization using interactive maps and charts
## Getting Started
To get started, clone the repository and run the setup script to install the required dependencies.
